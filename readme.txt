
Project Description:
Alien Invaders (Java): A simplified rendition of the classic 'Space Invaders.'
This project showcases my Java programming skills in game development.
Defend against descending alien invaders, shoot for high scores, and explore the fundamentals of game mechanics.
Dive into the code to understand the implementation and play the game using the GUI after running the game.

Instructions:
1. To play the game, run the main method in the Display class
2. A GUI will pop up, use the arrow keys to move around to dodge the bullets
3. use the space bar to shoot at the aliens
